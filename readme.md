# Data Structure & Algorithms

A personal repo to track my DSA journey — implementations, problem solving, and competitive programming.

---

## 📂 Structure

### 🟨 LeetCode

Problems solved on [LeetCode](https://leetcode.com), organized by difficulty.

- 🟢 `LeetCode/easy/` — Easy problems
- 🟡 `LeetCode/medium/` — Medium problems
- 🔴 `LeetCode/hard/` — Hard problems

### 🔵 Codeforces

Problems solved via A2OJ curated sheets on [Codeforces](https://codeforces.com), organized by rating.

- ⭐ `Codeforces/<rating>/` — Problems grouped by rating (e.g. 800, 1000, 1200 ...)
